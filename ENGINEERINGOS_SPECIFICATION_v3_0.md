## Preamble

Every great technical civilization rests on a specification layer that precedes implementation.
TCP/IP precedes the internet. SQL precedes every database. POSIX precedes every operating system.

EngineeringOS is the specification layer for Knowledge Engineering.

Everything that follows — every component, every algorithm, every DSL construct, every schema — derives from this document. When implementation conflicts with this specification, this specification wins. When this specification is wrong, a Constitutional RFC amends it. Nothing else has that authority.
